# OpenWrt R9 Docker: Простая инструкция по настройке для новичков

**Автор: Lean**

### Шаг 1

Сначала войдите в терминал роутера через **SSH**.

### Шаг 2

Введите команду:  
`/etc/docker-web`  
Эта команда автоматически скачает, установит и запустит веб-панель управления Docker (Portainer).

### Шаг 3

После успешного выполнения команды откройте браузер и введите:
`IP_вашего_роутера:9999`
Установите пароль администратора, выберите вариант **«local»** и войдите в интерфейс управления.

### Шаг 4

Нажмите на плитку **«local»** (с иконкой кита), чтобы перейти к управлению локальными контейнерами Docker.

---

**Примечание:**
Начиная с этой версии, если вы используете образы прошивок **x64 IMG** для обновления, раздел Docker будет сохраняться при условии, что структура разделов не меняется. Все ваши данные и программы внутри Docker останутся нетронутыми после обновления прошивки.
