[b]Прошивка OpenWrt 24.10.5 для CMCC RAX3000M (eMMC) от iqubik (v1.24)[/b]

Приветствую всех владельцев CMCC RAX3000M с eMMC-памятью!
Представляю вашему вниманию свежую, преднастроенную сборку прошивки, созданную для тех, кто хочет получить от роутера максимум. Каждый элемент, от прав доступа для Samba и Transmission до тонкостей работы системных скриптов, был тщательно настроен, чтобы [b]сэкономить вам часы ручной настройки[/b] и предоставить мощную, стабильную систему "из коробки".

Сценарий использования — "роутер для гиков": с удобным управлением и готовый к работе в качестве домашнего сервера.

[b]Ключевые особенности сборки:[/b]
[list]
[*] [b]База и экосистема пакетов:[/b] В основе лежит OpenWrt 24.10.5, но с подключенными репозиториями от ImmortalWrt и fantastic-packages. Это дает вам огромный выбор совместимого ПО, сочетая стабильность и новейшие пакеты.
[*] [b]Продуманная работа с eMMC:[/b] Прошивка идеально подходит для встроенной памяти. При первой загрузке запускается автоматический скрипт, который делает всю грязную работу за вас.
[spoiler="Как работает автоматическая разметка диска?"]
Скрипт (`99-extroot-setup.sh`) работает по принципу "запустил и забыл":
[list=1]
[*] [b]Инструмент:[/b] Использует надежную утилиту `sgdisk` для работы с таблицей разделов GPT.
[*] [b]Архитектура:[/b] Создает 3 раздела:
    [list]
    [*] `p6` (4 ГБ, ext4) для [b]extroot[/b] — здесь будет жить сама система (конфиги и пакеты).
    [*] `p7` (всё остальное место, ext4) для [b]данных[/b] — монтируется в `/mnt/data` для Docker, торрентов и ваших файлов.
    [*] `p8` (2 ГБ) для [b]swap[/b] (раздел подкачки).
    [/list]
[*] [b]Надежность:[/b] Скрипт идемпотентный — если он обнаружит, что система уже работает с extroot на разделе `p6`, он просто завершит работу, ничего не сломав. Это защищает от проблем при повторном запуске.
[/list]
[/spoiler]
[*] [b]Готовый домашний сервер:[/b]
    [list]
    [*] [b]Контейнеры:[/b] Установлен и преднастроен Docker. Все данные контейнеров хранятся на большом разделе `/mnt/data/docker`.
    [*] [b]Загрузки:[/b] Установлены и настроены Transmission и Aria2 (с веб-интерфейсом Ariang). Права доступа к папке `/mnt/data/downloads` уже настроены, что избавляет от головной боли.
    [*] [b]NAS и Медиа:[/b] Настроен SMB-сервер (ksmbd) и DLNA-сервер (minidlna). Доступ к файлам по сети работает сразу после старта.
    [*] [b]Система мониторинга:[/b] Cервис `luci_statistics` преднастроен для сбора важной статистики: пинг до внешних серверов, свободное место на разделах, температура процессора, загрузка CPU и памяти.
    [/list]
[*] [b]Улучшенный интерфейс (Web и SSH):[/b]
    [list]
    [*] [b]LuCI:[/b] Красивая тема Argon, полная русификация и набор "быстрых команд" в системном меню для административных задач в один клик.
[spoiler="Подробнее о быстрых командах"]
[list]
[*] [b]Перезапуск всех сетей:[/b] Полностью перезапускает сетевую подсистему роутера.
[*] [b]Показать порты:[/b] Выводит список всех портов, которые "слушает" роутер.
[*] [b]Докер очистка:[/b] Удаляет неиспользуемые образы, контейнеры и сети Docker.
[*] [b]Рестарт всех контейнеров:[/b] Перезапускает все запущенные Docker-контейнеры.
[*] [b]Топ жрунов оперативы:[/b] Показывает 5 самых ресурсоемких процессов.
[*] [b]Установить Portainer:[/b] Запускает скрипт для автоматической установки/обновления Portainer.
[*] [b]Поиск зомби процессов:[/b] Ищет и выводит "зависшие" процессы.
[*] [b]Мой внешний ip и страна:[/b] Показывает ваш текущий внешний IP-адрес и страну.
[*] [b]Докер аппетиты:[/b] Выводит подробную статистику по Docker.
[*] [b]Нагрузка на устройство:[/b] Отображает текущую нагрузку на процессор и систему.
[/list]
[/spoiler]
    [*] [b]Консоль:[/b] При входе по SSH вас встречает информативный `fastfetch` с состоянием системы, а в окружении доступны полезные алиасы для ускорения работы.
[spoiler="Подробнее о консольных улучшениях"]
[list]
[*] [b]Безопасный fastfetch:[/b] Скрипт в `/etc/profile` проверяет, является ли сессия интерактивной. `fastfetch` запускается только при входе в консоль и не мешает работе SFTP-клиентов.
[*] [b]Фикс кириллицы:[/b] В `.bashrc` прописана локаль `en_US.UTF-8`, что решает известную проблему с некорректным отображением русских букв в консоли.
[*] [b]Полезные алиасы (шорткаты):[/b]
    [list]
    [*] `update`, `upgrade-list`: для быстрой работы с пакетами.
    [*] `netrestart`, `netedit`, `wifi-reload`: для управления сетью.
    [*] `df`, `free`: для просмотра информации о системе.
    [*] `ll`, `..`, `...`: для удобной навигации.
    [*] "Безопасные" `rm`, `cp`, `mv`, которые просят подтверждения.
    [/list]
[/list]
[/spoiler]
    [*] [b]Живая LED-индикация:[/b] Прошивка использует светодиоды для наглядной обратной связи. Синий индикатор загорается, когда есть доступ в интернет. А зеленый — пульсирует в ритме "сердцебиения" системы, учащаясь при увеличении нагрузки (heartbeat).
    [/list]
[*] [b]Продвинутая сеть:[/b]
    [list]
    [*] [b]Оптимизация производительности (irqbalance):[/b] Включен сервис, который автоматически распределяет нагрузку от сетевых прерываний по всем ядрам процессора. Это предотвращает "затыки" на одном ядре и помогает достичь максимальной скорости при работе с гигабитным интернетом.
    [*] [b]Автоматическая настройка портов (UPnP):[/b] Включен и настроен сервис UPnP, который позволяет приложениям (например, на игровых приставках) автоматически пробрасывать нужные порты.
    [*] [b]Приватный DNS:[/b] Настроен локальный DNS-прокси, который повышает скорость и конфиденциальность ваших DNS-запросов через DNS-over-HTTPS.
    [*] [b]Fullcone NAT:[/b] Включен для лучшей совместимости с играми и P2P-приложениями.
    [*] [b]Умный роуминг (usteer):[/b] Включены стандарты 802.11k/v/r для более плавного переключения Wi-Fi клиентов. Для желающих тонко настроить пороги переключения, файл конфигурации `/etc/config/usteer` подробно прокомментирован на русском языке.
    [*] [b]Поддержка туннелей:[/b] Включены нужные пакеты и модули ядра, но прошивка не является готовым решением для обхода блокировок. Всё нужное каждый вправе настроить себе сам.
    [/list]
[/list]

[b][color=orange]ВАЖНО: Подготовка диска перед прошивкой![/color][/b]
[spoiler="Если у вас нестандартная разметка диска"]
Скрипт автоматической разметки в прошивке рассчитан на работу со стандартной (стоковой) таблицей разделов GPT. Если вы ранее экспериментировали с разметкой, у вас могут быть "лишние" разделы (`p6`, `p7` и т.д.), которые помешают работе скрипта.
[b]Перед прошивкой вы должны очистить их![/b] Есть два способа:

[b]Способ 1: Интерактивный (через cfdisk)[/b]
[list=1]
[*] Подключитесь к роутеру по SSH (на текущей прошивке) и выполните команду `cfdisk /dev/mmcblk0`.
[*] Внутри `cfdisk` удалите все разделы, идущие после `p5` (`mmcblk0p5`). Стоковые разделы (`p1`-`p5`) трогать не нужно.
[*] Сохраните изменения (Write) и выходите (Quit).
[/list]

[b]Способ 2: Прямая запись GPT (через dd)[/b]
Этот способ быстрее, но требует внимания. Он полностью перезаписывает таблицу разделов в начале диска, возвращая ее к заводскому состоянию.
[list]
[*] Загрузите файл `...-gpt.bin` из архива с прошивкой в `/tmp` на роутере.
[*] Выполните команду, [b]внимательно проверив имя файла[/b]:
[code]dd if=/tmp/openwrt-24.10.5-iqubik-v1.24-mediatek-filogic-cmcc_rax3000m-emmc-gpt.bin of=/dev/mmcblk0 bs=512 seek=0 count=34 conv=fsync[/code]
[/list]
Только после одного из этих шагов можно прошивать данную сборку. Если этого не сделать, скрипт разбиения диска просто не сработает.
[/spoiler]

[b][color=red]ВАЖНО: Первичная настройка безопасности![/color][/b]
Прошивка сделана для максимального удобства первого запуска. После входа в LuCI, пожалуйста, выполните следующие шаги:
[list=1]
[*] [b]Установите пароль на Wi-Fi:[/b] По умолчанию сеть `rax3000emmc` открыта. Зайдите в "Сеть" -> "Беспроводная сеть", нажмите "Редактировать" для каждой сети и на вкладке "Защита беспроводной сети" выберите WPA2/WPA3 и задайте надежный пароль.
[*] [b]Задайте пароль для SMB:[/b] По умолчанию для доступа к сетевым папкам используется логин `root` без пароля. Чтобы задать пароль, подключитесь к роутеру по SSH и выполните команду:
[code]ksmbd.adduser -a root[/code]
Система попросит вас ввести и подтвердить новый пароль. После этого доступ без пароля будет отключен.
[*] [b]Настройте доступ к Transmission:[/b] Веб-интерфейс торрент-клиента доступен в локальной сети без пароля. Если в вашей сети могут быть посторонние, ограничьте доступ.
[/list]
